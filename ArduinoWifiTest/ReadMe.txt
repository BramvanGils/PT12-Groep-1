Deze map bevat het testprogramma voor wifi verbinding maken met een Arduino en 
het query'en van SQL databases en het ophalen van results van queries, genaamd "WifiTestForm.cs", 
de class "SqlVerbinding.cs",die methods bevat om verbinding te maken en te communiceren in C# met een SQL database,
en de class "ArduinoWifi.cs, om via wifi verbinding te maken en te communiceren met een Arduino in C#.
